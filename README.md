# The Alchemy Landing Page

This is a Next.js landing page recreating "The Alchemy" trading course layout.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Responsive two-column layout
- YouTube video embed (placeholder)
- Payment form structure ready for Stripe integration
- Modern design matching the original screenshot
